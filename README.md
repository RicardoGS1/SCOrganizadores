## Stack
  - Kotlin
  - Clean + MVVM
  - Retrofit
  - Room (Cache)
  - Koin (DI)
  - Coroutines + Flow
  - Glide (Image loading)
  - Navigation Component
  - Airbnb Epoxy (Recycler view adapter replacement)
  - Facebook Shimmer (Displays loading state)
